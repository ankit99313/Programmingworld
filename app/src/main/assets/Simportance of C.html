Importance of C

Program written in C are efficient due to several variety of data types and powerful operators.

C is highly portable. This means that C programs written for one computer can be run on another with little or no modification.

Another important feature of C is its ability to extend itself.A C program is basically collection of functions that are supported by the C library.We can continously add our own functions to C library. With the availability of large number of functions, the programming task becomes simple.
