Q. What is structure and union ?

Ans. structure

A structure is a user defined data type. A structure creates a data type that can be used to group items of possibly different types into a single type.

struct book { char title[20];
char author[15]; int pages; float price; };

union

unions use the same concept as structure . However major difference between them in the uses of storage. In structures, each member has its own storage location, whereas in union all members use the same location.

union book;
{
char title[20];
char author[15];
int pages;
float price;
}
