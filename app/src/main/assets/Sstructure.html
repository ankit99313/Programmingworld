Structure

Structure is user defined date type,which can store the different data type variable together.

Example:-
struct student
{
char name[25];
int age;
char branch[10];
char gender;
};

union

union are a concept borrowed from structure and therefore follow the same syntax as structures.However, there is major distinction between them in terms of storage. In structures, each member has its own storage location, whereas all the member of a union use the same location.

union student
{
char name[25];
char author[15];
int pages;
float price;
};
