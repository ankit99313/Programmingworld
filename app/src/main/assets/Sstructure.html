Structure

Structure is user defined date type, which can store the different data type variable together.

Example:-
struct student
{
char name[25];
int age;
char branch[10];
char gender;
};

union

unions use the same concept as structure. However major difference between them in the uses of storage. In structures, each member has its own storage location, whereas in union all members use the same location.

union student
{
char name[25];
char author[15];
int pages;
float price;
};
