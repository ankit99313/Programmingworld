Fuction

Function is a collection of code that performs a specific task. Every C program has at least one function, which is main().
C function can be classified into two categories, standard library function and user defined function.

Standard library function

printf() and scanf() belong to the category of standard library function. These functions are defined in header files.

user defined function

We can also define functions as per our need. Such functions created by the user are known as user defined functions.
