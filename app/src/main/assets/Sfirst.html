#include

It is a preprocessor used to paste code of given file into current file. It is used to include system defined and user-defined header files into current. file.

<stdio.h>

stdio.h is a header file for standard input and output. Without this header file, one cannot display the results to the users on the screen or cannot input the values through the keyboard. It keeps the definition of predefined function like printf, scanf.

void

It is a return type. Void return types indicate that function doesn't return a value.

main()

Every C program has at least one function, which is main(). This function is called the entry point of any program. Any program starts from the main function.

printf()

It is a predefined function which is used to print the "character, string, float, integer, octal and hexadecimal values" on to the output screen.
