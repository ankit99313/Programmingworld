Data type

C language is rich in its data types. ANSI C provides three types of data types:

1.primary(Built-in) data types.

int:- Used to denote an integer type.
char:- Used to denote a character type.
float, doube:- Used to denote a floating point type.

2.Derived data types

Array, pointer.
We will discuss about it in later.

3.User defined data types

Structure, Union and Enumeration.
