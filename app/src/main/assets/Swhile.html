while loop

Syntax of while loop:

while(Condition)
{
statement-block;
}

(i)The while is an entry-controlled loop statement.

(ii)First condition is checked, and if the condition is true then body of the loop is executed.

(iii)After execution of the body, the test-condition is once again checked and if it is ture, the body is executed once again.

(iv)The process of repeated execution of the body continues until the test-condition finally becomes false and the control is transferred out of the loop.

#include <stdio.h>
void main()
{
int i=1;
while(i<=5)
{
printf("%d\n", i);
i++;
}
}
