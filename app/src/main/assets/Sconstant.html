Constant

Constant in C refer to fixed values that do not change during the execution of a program.C supports several types of constant.

Types of constants

Integer constants

1.Decimal:-45, -55, 5 etc.
2.Octal:-045, 0578 etc.
3.Hexadecimal: 0x23A, 0Xb4C, 0xFEA etc.

Real constant

Real constant are used to represent number containing fractional parts like 17.548.Such numbers are called real (or floating point) constants.Further examples of real constants are: 0.0083, -0.75, 435.36, +247.0

Single character constants

A single character constant(or simply constant) contains a single character enclosed within a pair of single quote marks.
'5', 'x', ';'

Note that the character constant '5' is not the same as the number 5.

String constants

A string constant is a sequence of character enclosed in double quotes.The character may be letters, numbers, special character and blank space.
"Hello", "1987", "welcome"
