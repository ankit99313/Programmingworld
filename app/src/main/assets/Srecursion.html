Recursion

Calling of function by itself until some specified condition is not satisfied is called recursion.

return-type function()
{
//body
function()
}

Example:-

#include <stdio.h>
int natural(int);
int main()
{
int n = 1;
printf(" The first 20 natural numbers are :");
natural(n);
return 0;
}
int natural(int n)
{
if(n<=20)
{
printf("%d ", n);
natural(n+1);
}
}
