Q. What are entry control and exit control loops ?

Ans. Entry control loop: In entry control loops first conditon is checked, and if the condtition or expression is true then control transfer into the body of the loop. Examples of exit controlled loop include, For Loop and While Loop.

Exit control loop: In exit control loop, loop body will be executed first and then condtion is checked and if condition is false then loop body will be executed once. If conditon is true then control will enter again into the loop. An example of exit controlled loop is Do While Loop.
