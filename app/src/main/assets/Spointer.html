pointer

pointer in C language is a variable that store/points the address of another variable. The pointer variable might be belonging to any of the data type such as int, float, char, double, short etc. Example are: int*p; char*p

where * is used to denote that "p" is pointer variable not a normal variable.

#include <stdio.h>
{
int var=20;
int *p;
p=&var;
printf("Address of var variable:%x\n", &var);

printf("Address stored in p variable:%x", p);
}

Address of var variable:60fef8
Address store in p variable:60fef8
