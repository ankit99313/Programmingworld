If

The if statement is a powerful decision making statement and is used control the flow of execution of statement. The general form of if statement is

if(condition)
{
statement-block;
}
statement-y;

The 'statement-block' may be a single statement or more then one statement. If the condition is true, the 'statement-block' will be executed; otherwise 'statement-block' will be skipped and control will jump to the 'statement-y'. If the condition is true both the 'statement-block' and the 'statement-y' are executed in sequence.

If-else

The if-else statement is an extension of the simple if statement. The general form is

If(condition)
{
True-block statement;
}
else
{
False-block statement;
}
statement-y;

If the condition is true, then the 'True-block statement' will be executed; otherwise, the 'False-block statement' will be executed. At a time either 'True-block statement' or 'False-block statement' will be execute, not both.
