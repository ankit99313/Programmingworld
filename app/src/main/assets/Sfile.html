File handling

A file is a place on the disk where a group of related data is stored. C supports a number of function that have the ability to perform basic file operation, which include
1.naming a file,
2.opening a file,
3.reading data from a file,
4.writing data to a file,and
5.closing a file

Function name operation
fopen() open a text file
fclose close a file which has been opened for use
fprintf() write a set of data values to a file
fscanf() reads a set of data values from a file
getc() reads a character from a file
putc() write a character to a file
getw reads an integer from a file
putw writes an integer to a file

Following is the general format for some operation on a file

operation task
opening a file File *fp;
fp=fopen("filename","mode");
closing a file File *fp;
fclose(fp);
mode of operation

mode task
r It opens an existing text file for reading purpose.
r+ It opens an existing text file for reading and writing purpose.
w It opens a text file for writing purpose.
w+ It opens a text file for writing and reading purpose.
