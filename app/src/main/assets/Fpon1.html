Q. What is difference between array and pointer ?

Ans. (i)An array is a collection of elements of similar data type whereas the pointer is a variable that stores the address of another variable.

(ii)An array size decides the number of variables it can store whereas; a pointer variable can store the address of only one variable in it.

(iii)Arrays can be initialized at the definition, while pointers cannot be initialized at the definition.

(iv)Arrays are allocated at compile time while pointers are allocated at the runtime.

(v)Arrays are static in nature which means once the size of the array is declared, it cannot be resized according to users requirement. Whereas pointers are dynamic in nature, which means the memory allocated can be resized later at any point in time.
