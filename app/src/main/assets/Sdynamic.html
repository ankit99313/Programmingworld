Dynamic memory allocation

By the using of dynamic memory allocation you can allocate memeory at runtime .There are four functions are used to allocating and freeing memory during execution of a program.
(i)malloc()
(ii)calloc()
(iii)realloc()
(iv) free()

malloc():Allocates requested size of bytes and returns a pointer first byte of allocated space.

calloc(): Allocates space for an array of elements, initialize them to zero and then returns a pointer to the memory.

free(): Free previously allocated space.

realloc(): Modifies the size of previously allocated space.

malloc

Syntax:
ptr = (data-type*) malloc(n*size0f(data-type))

For example:
ptr=(int*)malloc(100* sizeof(int));

Since the size of int is 4 bytes, this statement will allocate 400 bytes of memory. And, the pointer ptr holds the address of the first byte in the allocated memory.

calloc

Syntax:
ptr = (data-type*) calloc(n, sizeof(data-type))

For example:
ptr = (float*) calloc(25, sizeof(float));

The malloc() function allocate memory and doesn't initalize the memory. Whereas, the calloc() function allocates memory and initialize allocted memory to zero.

realloc

Syntax:
ptr = realloc(ptr, x);

If the memory allocated by malloc() is less or more than required, you can resize the previously allocated memory using the realloc() function.

free

Syntax:
free(ptr);

free() function is use for release those spaces which is created by malloc() or calloc().

This statement frees the space allocated in the memory pointed by ptr.
