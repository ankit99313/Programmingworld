Enumeration

(i)In C language enumeration is a user defined datatype.

(ii)It is used to assign names to the integral constants which makes a program easy to read and maintain.

(iii)The keyword "enum" is used to declare an enumeration.

Syntax:
enum enum-name {const1, const2,........};

Example:

enum day{Mon, Tue, Wed, Thur, Fri, Sat, Sun};
int main()
{
printf("The value of enum day:%d %d %d %d %d %d %d", Mon, Tue, Wed, Thur, Fri, Sat, Sun);
}

The value of enum day:0 1 2 3 4 5 6
