Operator

It is a special symbol which is used to perform logical or mathematical operation on data or variable.

Types of operator:

1.Arithmetic operator:-

Symbol Meaning
+
Addition
-
Subtraction
*
Multiplication
/
Division
%
Modulus

(i)Used to perform arithmetic operations.

(ii)They have high priority.


2.Relational operators

Symbol Meaning
==
Equal to
!=
Not equal to
>
Greater than
<
Less than
>=
Greater than or equal to
<
Less than or equal to

(i)These operators are used to compare two values.

(ii)Used to form conditions in programming.

(iii)They have lower priority than arithmetic operators.


3.Logical operators

Symbol Meaning
&&
Logical AND
||
Logical OR
!
Logical NOT

(i)It is used to combine two or more conditions logically.

(ii)They have lower priority than relational operatos.

Logical 'AND' operator:

(i)It is denoted by symbol '&&'.

(ii)Logical And is true if all the conditions are true.

Logical 'OR' operator:

(i)It is denoted by '||'.

(ii)It always returns boolean value(True/False). It returns true if either one or both conditon is true or otherwise it will return false.

Logical 'NOT' operator:

(i)It is denoted by symbol '!'.

(ii)It is true if condition is false.

4.Assignment operators

=, +=, -=, *=, /=, %=

5.Bitwise operators

Symbol Meaning
&
Bitwise AND
!
Bitwise OR
<<
Shift left
>>
Shift right

6.Increment/Decrement operators

Symbol Meaning
++
Increment
--
Decrement

(i)These operators can't be operators on constant values.

(ii)Both operators can be used in prefix(++x) and postfix(x++).

Increment operator:

(i)It is denoted by symbol '++'.

(ii)It is an unary operator.

(iii)It increments the value of operand by one(1).

Decrement operator:

(i)It is denoted by symbol '--'.

(ii)It is an unary operator.

(iii)It decrements the value of operand by one(1).

7.Special operators

Symbol Meaning
&
It is used for find address of a variable.
*
It is used to declare pointer type variable.
sizeof()
It returns the memory size of variable.
